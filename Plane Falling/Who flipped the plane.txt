You're touching on an important distinction.

Nobody is saying 0 ≠ 0.

In fact:

0 = 0
1 = 1

are perfectly valid statements.

The problem appears when you try to interpret percentages and division involving zero.

For example:

100% × 0 = 0

That's completely valid.

But:

100% = x / 0

is not valid mathematics because percentages are ratios.

100% means:

1.0 = x / y

For 100%, the numerator and denominator are equal:

1 = 5/5
1 = 10/10
1 = 100/100

But if the denominator becomes zero:

x / 0

the expression stops being defined.

Here's why.

Suppose:

1 = x / 0

Multiply both sides by 0:

1 × 0 = x
0 = x

So you'd get:

1 = 0/0

Now try:

2 = x / 0

Multiply by 0:

2 × 0 = x
0 = x

which gives:

2 = 0/0

And similarly:

3 = 0/0
1000 = 0/0
-7 = 0/0

If division by zero were allowed, every number would equal every other number.

That's why mathematics doesn't say:

"0 doesn't equal 0."

Instead it says:

"Division by zero is undefined because it destroys the uniqueness of numbers."

"Everybody survived."

In logic, that's called vacuous truth. It creates some very strange-looking results, but it still doesn't imply that zero becomes one.

Your intuition is probing the boundary between arithmetic, set theory, and logic. That's exactly where a lot of famous paradoxes live. The catch is here:

100% × 0 = 0

True.

Statement 2
100% = x / 0

Not valid in ordinary arithmetic.

Those two statements look similar but are fundamentally different.

Let's use your plane example.

Suppose:

Plane-falling events = 0
Deaths = 0
Survivors = 0

You are tempted to say:

survival rate = 100%

because nobody died.

But another person could say:

survival rate = 0%

because nobody survived either.

And another could say:

survival rate = 50%

because there were no participants at all.

The reason mathematics refuses to assign a percentage is because all three interpretations are equally unsupported by the data.

The data only tells us:

0 survivors
0 deaths
0 participants

Nothing more.

What's interesting is that you're very close to a real concept in mathematics called the empty set.

For example:

All unicorns on Mars are blue.

Is that true or false?

There are no unicorns on Mars.

Classical logic says the statement is true, because there is no counterexample.

This feels strange because truth is being assigned to a situation involving zero objects.

That's similar to your intuition:

If 100% of the participants survived, but there were zero participants, does that somehow create a "one"?

Mathematically, no.

Philosophically, you're noticing that completeness and quantity are different things.

Quantity asks: "How many?"
Completeness asks: "How much of the set satisfies the condition?"

When the set is empty:

quantity = 0

but

completeness

becomes a strange question because there is no set to measure.

So the paradox you're feeling isn't that 0 doesn't equal 0.

It's that:

An empty category can still satisfy a logical condition.

That's one of the reasons empty sets, vacuous truths, and division by zero feel so unintuitive. They sit right at the boundary between counting things and reasoning about things.
